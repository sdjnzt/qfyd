# 曲阜远东职业技术学院云定制化平台

基于 Vue3 + TypeScript + Element Plus 的曲阜远东职业技术学院云定制化平台前端项目。

## 项目概述

本项目部署在客户侧机房，通过云定制化项目，全面优化教学办公流程，实现自动化办公流程，满足学校业务动态需求，为一站式综合服务平台做好数据存储及安全防护。

## 云管平台组成

- **cSight软件**：提供全面的云基础设施管理和监控能力
- **教学管理平台**：4套独立系统，支持课程、学生、教师和考试全流程管理
- **设备管理许可**：
  - 网络设备：50台管理许可，包括路由器、交换机、防火墙等
  - 服务器：30台设备管理许可，支持物理服务器和虚拟服务器
  - 存储设备：20台管理许可，支持多种存储类型

## 云平台核心功能

- **统一监控面板**：实时监控所有资源状态和性能
- **资源可视化管理**：教学系统、网络设备、服务器、存储设备的集中管理
- **告警管理**：多级告警策略，确保系统稳定运行
- **许可管理**：集中管理各类资源许可证，提高资源利用率
- **自动化运维**：减少人工干预，提高系统可靠性
- **数据备份与恢复**：保障数据安全

## 平台模块

### 教学管理模块
- 课程资源：教学资料、课件管理
- 教学计划：教学安排、课表管理
- 学生管理：学生信息、成绩管理
- 教师管理：教师信息、工作量管理

### 基础设施管理模块
- 网络管理：路由器、交换机、防火墙等设备管理
- 服务器管理：物理和虚拟服务器资源监控与管理
- 存储管理：存储设备监控与容量管理
- 监控告警：设备状态监控、故障告警
- 备份管理：数据备份策略与恢复

### 办公自动化模块
- 流程审批：教学、行政审批流程
- 文档管理：公文、教案管理
- 资源调度：教室、设备等资源预定

### 安全防护模块
- 数据备份：定期备份关键数据
- 权限控制：多级权限设计，保障数据安全
- 安全审计：操作日志记录与审计

## 技术栈

- ECharts

## 开发环境要求

- Node.js >= 16
- npm >= 7

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 项目结构

```
src/
  ├── api/          # API 接口
  ├── assets/       # 静态资源
  ├── components/   # 公共组件
  ├── router/       # 路由配置
  ├── store/        # 状态管理
  ├── styles/       # 全局样式
  ├── utils/        # 工具函数
  ├── views/        # 页面组件
  │   ├── cloud-management/ # 云管平台组件
  │   ├── dashboard/       # 仪表盘组件
  │   ├── server/         # 服务器管理组件
  │   ├── network/        # 网络管理组件
  │   ├── storage/        # 存储管理组件
  │   ├── monitor/        # 监控组件
  │   ├── backup/         # 备份管理组件
  │   ├── alarms/         # 告警管理组件
  │   └── ...
  ├── App.vue       # 根组件
  └── main.ts       # 入口文件
```
